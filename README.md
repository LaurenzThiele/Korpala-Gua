# Korpala Gua — Database Gua Indonesia

Platform eksplorasi dan dokumentasi gua-gua Indonesia oleh **Korpala Universitas Hasanuddin**.

**Live:** [korpala-speleologi.netlify.app](https://korpala-speleologi.netlify.app)

---

## Tentang

Korpala Gua adalah platform pemetaan gua interaktif yang mendokumentasikan lokasi, karakteristik, dan data eksplorasi gua-gua di seluruh Indonesia. Dibangun untuk mendukung kegiatan penelitian, eksplorasi, dan pelestarian kawasan karst Nusantara.

## Fitur

- **Peta interaktif** — Visualisasi lokasi gua di seluruh Indonesia dengan OpenTopoMap
- **Database gua** — Tabel lengkap dengan filter, pencarian, dan pengurutan
- **Detail gua** — Informasi kedalaman, koordinat UTM, wilayah geografis, jenis gua, dan peta/dokumen terlampir
- **Panel admin** — Tambah, edit, dan hapus data gua dengan autentikasi aman
- **Upload dokumen** — Simpan peta gua dalam format PNG, JPEG, PDF, atau HTML
